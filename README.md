# Roboìmọ́tótó_Mobile_smart_bin
Roboìmọ́tótó is an autonomous smart bin designed to tackle waste-related disease transmission in Africa. Using an Arduino Uno, IR sensors for patrol, and HC-05 for voice-activated lid control, it eliminates physical contact with waste surfaces. It is a locally-producible, smart city solution for hospitals, campuses, and markets.
